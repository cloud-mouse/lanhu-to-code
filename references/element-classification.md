# 元素分类与勾选清单详细指南

## 元素分类规则

对设计稿中每个可见元素，判断其实现方式：

| 分类 | 判断依据 | 实现方式 | 示例 |
|------|---------|---------|------|
| **DOM 文字** | 纯文本内容，有明确字体/字号/颜色 | HTML 标签 + CSS | 标题、正文、价格数字、状态文字 |
| **DOM 形状** | 简单几何形状（矩形/圆角矩形/圆形/线条），纯色或渐变填充 | HTML 标签 + CSS | 卡片背景、分割线、进度条、圆角按钮、渐变背景块 |
| **切图-图标** | 小尺寸（< 100px）、有明确语义的图形 | `<image>` + 切图文件 | 返回箭头、勾选图标、锁定图标、步骤图标 |
| **切图-装饰** | 复杂的装饰性图形，无法用 CSS 实现 | `<image>` + 切图文件 | 头部大气背景图、波浪分割线、装饰花纹 |
| **切图-背景** | 覆盖大面积区域、含丰富色彩/图案的图片 | `<image>` + 切图文件 + 背景色兜底 | hero 区域背景、卡片装饰纹理 |

## 分类判定决策树

```
该元素是文字吗？
├─ 是 → DOM 文字
└─ 否 → 该元素能用纯 CSS 实现（矩形/圆角/渐变/阴影）吗？
    ├─ 能 → 该元素是纯色/渐变的简单几何形状吗？
    │   ├─ 是 → DOM 形状
    │   └─ 否（含纹理/图案/复杂曲线）→ 切图-装饰
    └─ 不能 → 该元素尺寸 < 100px 且有明确语义吗？
        ├─ 是 → 切图-图标
        └─ 否 → 切图-背景/装饰
```

## 勾选清单输出格式

```
## 元素勾选清单

### 页面级
- [ ] 页面背景色: [值] → DOM 形状

### 区域: [区域名称]
- [ ] [元素描述] → 分类 | 关键属性摘要

### 例如：
### 区域: Hero 头部
- [ ] 背景装饰图 (750×400) → 切图-背景 | hero_bg.png
- [ ] 返回按钮图标 (24×24) → 切图-图标 | icon_back.png
- [ ] "邀好友得奖励" → DOM 文字 | 80rpx, #FFFFFF, Bold
- [ ] 副标题背景条 → DOM 形状 | #FFEDD8, 圆角12rpx
- [ ] "邀请好友注册..." → DOM 文字 | 26rpx, #946B41
- [ ] "规则"按钮 → DOM 形状 | rgba(255,255,255,0.2), 圆形
```

## 切图资源处理

对分类为"切图"的元素：

1. **下载切图**到项目静态资源目录（如 `src/static/images/<page>/`）
2. **检查切图可用性：**
   - 图片尺寸是否合理（不过大也不过小）
   - 格式是否正确（PNG 透明底 / JPG 有背景）
   - 是否需要适配 @2x/@3x
3. **命名规范：** `{区域}_{用途}.{ext}`，如 `hero_bg.png`、`icon_back.png`、`step_invite.png`
4. **背景图兜底：** 切图-背景 类资源加载前应有背景色兜底，避免白屏

## 切图背景分析（防止 CSS 尺寸错误）

**下载切图后，必须判断切图是否包含不透明背景，这决定了 CSS 中的尺寸和背景设置。**

### 判断方法

基于 slices 数据中的类型字段：

| 切图类型 | 含义 | CSS 写法 |
|---------|------|---------|
| `artboard` 类型 | 完整组件切片，**含背景** | `width: N rpx; height: N rpx;` 不加额外背景 |
| `shapeLayer` 类型 | 纯图形切片，**不含背景** | 外层容器设背景色 + 尺寸，内层图标缩小 |

### 分析示例

```
切图 A: 品牌动态 (artboard, 80x80, 含 Rectangle 1080 #FFF4E6 背景)
→ 含背景 ✓
→ CSS: width: 80rpx; height: 80rpx; （直接全尺寸，不加 background-color）

切图 B: 返回箭头 (symbolInstence, 48x48, 纯箭头图形)
→ 不含背景 ✓
→ CSS: 外层 48rpx 容器, 内层 icon 48rpx 全尺寸（无需额外背景）
```

### 在勾选清单中标注

对每个切图元素，标注是否含背景：

```
- [ ] 步骤图标 (80×80) → 切图-图标 | icon_step_invite.png | 含背景 ✓
- [ ] 返回按钮 (48×48) → 切图-图标 | icon_back.png | 不含背景
- [ ] 进度勾选图标 (36×36) → 切图-图标 | icon_check.png | 含背景 ✓
```

**关键规则：禁止参考其他页面的 icon 尺寸写法。每个设计稿的切图内容可能不同，必须以当前切图的实际内容为准。**
